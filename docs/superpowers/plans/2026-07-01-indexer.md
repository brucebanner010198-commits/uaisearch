# Indexer Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stand up the self-hosted OpenSearch index that stores crawled page chunks with embeddings, and the `index_page()` pipeline that writes to it while skipping near-duplicates.

**Architecture:** A single `uaisearch` Python package. Pages arrive as `ExtractedPage` objects (produced by the crawler, built in a later plan — for this plan they're constructed directly in tests). `index_page()` chunks the page text, embeds each chunk locally, computes a per-page `domain_quality` heuristic, skips the page entirely if its simhash is a near-duplicate of something already indexed, and writes one OpenSearch document per chunk.

**Tech Stack:** Python 3.12, `opensearch-py` (OpenSearch client), `sentence-transformers` (local embeddings, no paid API), `simhash` (near-duplicate detection), `numpy`, `pytest`.

## Global Constraints

- No proprietary paid APIs for indexing — embeddings are generated locally via `sentence-transformers`, never a hosted embedding API.
- Dependencies pinned to exact versions in `pyproject.toml` (per project security baseline).
- `domain_quality` and `ad_ratio` are ranking signals only — never returned in API responses (enforced starting in the Retrieval/API plans; this plan just stores them).
- `index_page()` enforces the design doc's legal exclusion floor via a URL/domain blocklist (Task 8) — the operator-maintained mechanism for DMCA takedowns and known-illegal domains. Be precise about what this is and isn't: this pipeline only ever crawls and indexes *text*, never images, so there is no image content for perceptual-hash CSAM matching (PhotoDNA/PDQ-style, via NCMEC's ESP program or a vetted provider like Thorn Safer) to run against. If this system is ever extended to fetch or serve image content, integrating one of those real hash-matching services becomes a hard legal prerequisite before that extension ships — a URL/domain blocklist alone would not be sufficient at that point.

```python
# src/uaisearch/indexer.py
def chunk_text(text: str, size: int = 450, overlap: int = 50) -> list[str]:
    words = text.split()
    if not words:
        return []
    step = size - overlap
    chunks = []
    for start in range(0, len(words), step):
        chunks.append(" ".join(words[start:start + size]))
        if start + size >= len(words):
            break
    return chunks
```

```python
# append to src/uaisearch/indexer.py
from simhash import Simhash, SimhashIndex


def load_simhash_index(client: OpenSearch) -> SimhashIndex:
    # ponytail: loads every simhash into memory at once; re-load periodically
    # or switch to a sharded index if the corpus outgrows a single process's memory
    response = client.search(
        index=INDEX_NAME,
        body={"query": {"match_all": {}}, "_source": ["simhash"], "size": 10000},
    )
    objs = [(hit["_id"], Simhash(hit["_source"]["simhash"])) for hit in response["hits"]["hits"]]
    return SimhashIndex(objs, k=3)


def is_near_duplicate(index: SimhashIndex, simhash_value: int) -> bool:
    return len(index.get_near_dups(Simhash(simhash_value))) > 0
```

```python
# append to src/uaisearch/indexer.py
from uaisearch.models import ExtractedPage


def index_page(client: OpenSearch, page: ExtractedPage, dedup_index: SimhashIndex) -> int:
    if is_near_duplicate(dedup_index, page.simhash):
        return 0
    # ponytail: per-page heuristic; swap for a domain-level rolling average
    # if ad_ratio proves too noisy at the individual-page level
    domain_quality = round(1.0 - page.ad_ratio, 4)
    indexed = 0
    for chunk in chunk_text(page.text):
        client.index(index=INDEX_NAME, body={
            "url": page.url,
            "domain": page.domain,
            "title": page.title,
            "chunk_text": chunk,
            "embedding": embed(chunk),
            "ad_ratio": page.ad_ratio,
            "domain_quality": domain_quality,
            "crawl_date": page.crawl_date,
            "simhash": page.simhash,
        })
        indexed += 1
    dedup_index.add(page.url, Simhash(page.simhash))
    return indexed
```
